One-time setup required
1
Register a free Azure App — go to portal.azure.com → Azure Active Directory → App Registrations → New Registration. Set redirect URI type to Single Page Application (SPA) and enter your app URL.
2
Copy your Application (Client) ID from the app overview page and paste it below.
3
Upload Breaches2026.xlsx to OneDrive root. The file must have a table named BreachTable on a sheet named Breaches.
4
Enter your Client ID below and click Save & Connect. A Microsoft login popup will appear. After login the app syncs automatically.
